# 🌱 Smart Agriculture Using IoT

## Overview

Smart Agriculture Using IoT is a team-based project developed to automate irrigation using IoT technology. The system monitors important environmental parameters such as soil moisture, temperature, and humidity in real time. Sensor data is displayed on a web dashboard, allowing users to monitor field conditions and control the irrigation system efficiently.

## Features

- Real-time monitoring of soil moisture
- Temperature and humidity monitoring
- Live sensor data displayed on a web dashboard
- Automatic or manual water pump control
- Efficient water management for smart farming

## Hardware Components

- Arduino
- Soil Moisture Sensor
- DHT11/DHT22 Sensor
- Relay Module
- Water Pump
- Breadboard
- Jumper Wires
- Power Supply

## Working

The soil moisture sensor and DHT sensor collect real-time environmental data. The Arduino processes this information and sends it to a web dashboard, where users can view soil moisture, temperature, and humidity values. Based on the sensor readings, the water pump can be controlled through a relay module to irrigate the soil when required.

## Technologies Used

- Arduino IDE
- Embedded C (Arduino Programming)
- IoT
- Sensor Integration

## Team Project

This project was completed as part of an 8-member team. It involved hardware setup, sensor integration, system testing, and demonstrating how IoT can be used for smart irrigation and efficient water management.
